## ✨ Возможности

| Функция | Описание |
|---------|----------|
| 🔍 **Сканирование сетей** | Поиск, фильтрация, сортировка, экспорт CSV |
| 📊 **Мониторинг сигнала** | График в реальном времени, avg/min/max |
| 💾 **Профили Wi-Fi** | Просмотр, подключение, удаление, экспорт |
| 🔑 **Пароли сетей** | Отображение и копирование паролей |
| 🔗 **Подключение** | WPA2 / WPA3 / Open, перезапуск адаптера |
| 🌐 **Диагностика** | ping, tracert, сброс TCP/IP, DNS |
| 📱 **Мобильная версия** | React PWA с тёмной темой |

## 🚀 Запуск

```python
python wifi_manager.py
```

Требуется Python 3.10+ и Windows 10/11. Внешних зависимостей нет.

## 📁 Файлы

- `wifi_manager.py` — основное приложение
- `WiFiManagerMobile.jsx` — мобильная React версия
- `WiFiManagerPro_Report.pptx` — презентация
- `WiFiManagerPro_Report.docx` — технический отчёт
